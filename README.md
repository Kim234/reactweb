## 개요

> 국물 여부, 맵기 정도 등에 따라 식당 추천과 식당을 예약 및 평가할 수 있는 웹입니다.



## 주요 기술 
```sh
* Frontend
    - React (term_project_front 폴더)
* Backend
    - Node js (term_project_back 폴더)
* DB
    - Mongo DB
```


## Enviroment
```sh
* react = 17.0.1
* npm >= 6.14.11
* Node.JS = 14.16.0 
```


## Install

```sh
./term_project_front 폴더와 ./term_project_back 폴더에 들어간 후, 각각 다음과 같이 실행합니다.
npm install
```

## Usage 


```sh
./term_project_front 폴더와 ./term_project_back 폴더에 들어간 후, 각각 다음과 같이 실행합니다.
npm start
```
